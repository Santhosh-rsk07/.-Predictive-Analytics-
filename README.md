# Predictive Analytics Using Linear Regression

## Overview

This project demonstrates a basic Predictive Analytics model using Python and Machine Learning. It predicts student marks based on study hours using Linear Regression.

## Features

* Data visualization using scatter plots
* Linear Regression model training
* Marks prediction based on study hours
* Prediction trend visualization

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Requirements

```bash
pip install pandas matplotlib scikit-learn
```

## Usage

Run the script using:

```bash
python predictive_analytics_.py
```

## Dataset

The sample dataset contains:

* Study Hours
* Student Marks

## Output

* Scatter plot of study hours vs marks
* Trained Linear Regression model
* Predicted marks for 9 study hours
* Regression line visualization

## Learning Objectives

* Understand Predictive Analytics
* Learn Linear Regression basics
* Visualize data and predictions
* Build simple machine learning models
